

* @Service = 비즈니스 로직, 계산 수행, 외부 API 호출을 수행 하는 JAVA 클래스를 표시


* @Controller = URL 요청이 오면 응답 담당


---------------------------------------------------------------------------------------


* 작업 순위
   1. 클래스를 만들면 어노테이션을 먼저 만들기
      컨트롤러 클래스 = 컨트롤러 어노테이션
      서비스 클래스 = 서비스 어노테이션
   2. @Autowired 하기
      @Autowired 객체 생성과 소멸을 Spring이 관리 함
